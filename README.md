# Portfolio Aura Auvarose 🚀

Website portofolio pribadi yang interaktif, responsif, dan dinamis. Dibangun untuk menampilkan profil, perjalanan belajar, proyek, dan sertifikat, sekaligus mengimplementasikan integrasi AI secara langsung.

## ✨ Fitur Utama
* **Dark/Light Mode:** Transisi tema yang mulus dengan efek gelombang (*ripple*).
* **Integrasi AI (Gemini 2.0 Flash):** Asisten chatbot cerdas yang disematkan langsung untuk menjawab pertanyaan pengunjung.
* **Database Dinamis (Supabase):** Pengelolaan data Proyek, Sertifikat, dan Buku Tamu (Guestbook) secara *real-time*.
* **Admin Panel:** Dashboard khusus untuk menambah/menghapus konten portofolio tanpa harus menyentuh kode.
* **Responsive Design:** Tampilan yang dioptimalkan untuk Desktop, Tablet, dan Mobile.

## 🛠️ Tech Stack
* **Frontend:** Next.js (React), CSS Native
* **Backend:** Next.js Route Handlers
* **Database:** Supabase (PostgreSQL)
* **AI API:** Google Gemini API
* **Deployment:** Vercel yang gratis2 aja
